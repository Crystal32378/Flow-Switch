# URL Launcher MVP

This is the recommended free MVP.

It does not call any Web API. It stores four mode-to-URL mappings locally, opens the selected playlist or album URL, and lets the user's music app continue playback with its native behavior.

This project is not affiliated with, endorsed by, or sponsored by Spotify. Spotify is mentioned only because Spotify playlist and album URLs are supported inputs.

## Use

1. Double-click `4_manual_setup.command`.
2. Paste one playlist or album URL for each mode.
3. Double-click `5_manual_start.command`.
4. Pick a mode number and press Enter.
5. Press play in your music app if it does not start automatically.

`manual_playlists.json` is created locally and is ignored by git.
