# C_url_launcher_version

Free Spotify accounts can still use this version.

It does not call the Spotify Web API. It stores four mode-to-URL mappings locally, opens the selected Spotify playlist or album, and lets the Spotify app continue playback with its native autoplay behavior.

## Use

1. Double-click `4_manual_setup.command`.
2. Paste one Spotify playlist or album URL for each mode.
3. Double-click `5_manual_start.command`.
4. Pick a mode number and press Enter.
5. Press play in Spotify if it does not start automatically.

`manual_playlists.json` is created locally and is ignored by git.
